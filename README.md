# ATM

## Requirements

### Windows - MSYS2

To properly build the project on Windows, it is recommended to use the **MSYS2** environment. After installing it, open the **MSYS2 UCRT64** terminal and install the necessary packages:

```bash
pacman -S mingw-w64-ucrt-x86_64-cmake mingw-w64-ucrt-x86_64-gcc mingw-w64-ucrt-x86_64-gdb mingw-w64-ucrt-x86_64-doxygen
```
